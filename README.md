# twitter_sentiment_analysis_NLP
Detect hate speech in tweets

This is a guided project from 
https://courses.analyticsvidhya.com/courses/take/twitter-sentiment-analysis/texts/

Methods used and learned: 
- NLTK library
- Numpy, pandas, seaborn, matplotlib
- Features: Bag-of-words, TF-IDF
- Modelling: Logistic regression, SVM, Random Forest classification, XGboost classification
